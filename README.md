# 🎮 Kelime Tetris Oyunu

Kelime tabanlı bir Tetris oyunu. Klasik Tetris oyununun kelime bulmaca ile birleşmiş halidir.

## 🎯 Özellikler
- Kelime tabanlı Tetris mekaniği
- Modern ve responsive tasarım
- Mobil uyumlu
- Skor sistemi

## 🚀 Nasıl Oynanır?
1. **Başla** butonuna tıklayın
2. **Sol/Sağ ok** tuşları ile blokları hareket ettirin
3. **Yukarı ok** ile blokları döndürün
4. **Aşağı ok** ile hızlandırın
5. Harfleri birleştirerek **kelimeler oluşturun**
6. Her kelime için **puan kazanın**

## 🛠️ Teknolojiler
- **HTML5**
- **CSS3**  
- **JavaScript**
